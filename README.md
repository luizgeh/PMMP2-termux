# Instructions:

1. Create a directory using `mkdir (your directory)`
2. Go to directory using `cd (your directory)`
3. Install the package **wget** using `pkg install wget`
4. Run the bash file with `./install.sh` or `bash install.sh`
5. Run the server using `start.sh`
6. Setup the PocketMine-MP server

# Commands

- `mkdir (your directory)`
- `cd (your directory)`
- `pkg install wget`
- `./install.sh`
